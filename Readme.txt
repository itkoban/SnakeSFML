Q # Как запустить игру?
-------------------------------------------------- 
A # Запускай SnakeSFML.exe в папке Realese

--------------------------------------------------
Q # Как выйти из меню // войти в меню?
-------------------------------------------------- 
A # Нажимай ESC и будет твоё счатье ;)
--------------------------------------------------


--------------------------------------------------
Q # Какое управление в игре?
-------------------------------------------------- 
A # ESC в меню            ====> Выйти из меню
A # ESC	в игре            ====> Войти в меню
...
A # Стрелка вверх  ====> Повернуть вверх
...
A # Стрелка вправо ====> Повернуть вправо
...
A # Стрелка вниз   ====> Повернуть вниз
...
A # Стрелка влево  ====> Повернуть влево
--------------------------------------------------


--------------------------------------------------
